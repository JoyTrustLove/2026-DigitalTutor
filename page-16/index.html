▎ 품질관리 — 이중 게이트 체계
1차 게이트 — 주관기관 현장 모니터링
일일 체크리스트 출결·교육장 환경·기기 상태·네트워크·안전 사항을 매일 점검·기록
수강생 피드백 일일 간편 설문(5점 척도 + 자유 의견)으로 수강 만족도 실시간 수합
이슈 대응 로그 돌발 상황 발생 → 즉시 기록 → 당일 PM 보고 → 익일 조치 완료
주간 보고 운영 현황·출석률·이슈·개선 사항을 주간 보고서로 재단 제출
2차 게이트 — 협력기관 검토위원회·수업 관찰
검토위원회 교육 전(교안 검토)·교육 중(현장 점검) 최소 2회 운영, 개선 권고 발행
수업 관찰 협력기관 품질관리팀이 현장 수업 참관, 관찰 체크리스트 기반 평가
강사 피드백 수업 관찰 결과를 강사별 피드백 리포트로 작성·전달·개선 확인
환류 보고서 모니터링 결과 → 개선 사항 도출 → 다음 차시 반영 → 재검증 루프
▎ 리스크 관리 — 주요 리스크 및 대응 방안
위험도 리스크 항목 사전 예방 조치 발생 시 대응 담당
상 모집 인원 미달 120% 목표 모집, 4대 채널 동시 운영, D-14 조기 공고 추가 홍보 집중, 대기자 순차 연락, 모집 기간 연장 협의 주관
상 중도 이탈률 증가 사전 OT 통해 교육 일정·강도 안내, 동기 부여 프로그램 개별 상담, 이탈 사유 분석, 대기자 충원 주관
중 교육장 기기·네트워크 장애 사전 테스트 2회, 여분 기기 15% 확보, 이중 네트워크 구성 장비·안전팀 즉시 대응, 여분 교체, 모바일 핫스팟 가동 장비팀
중 강사 돌발 불참 강사풀 정원 대비 150% 확보, 대체 강사 사전 지정 대체 강사 즉시 투입, 온라인 전환 옵션 가동 주관+협력
하 현장체험 협력학교 미확보 교육청 공문 사전 협조, 다수 후보 학교 확보 교내 시뮬레이션 실습으로 대체, 영상 참관 병행 {확인필요} 협력
16면 핵심 요약
마스터 일정은 5월 착수 → 5~9월 과정 개발·모집 → 10~11월 본 교육 → 12월 성과확산·정산의 8개월 체계로 운영됩니다. 품질관리는 주관기관 현장 모니터링(1차 게이트)과 협력기관 검토위원회·수업 관찰(2차 게이트)의 이중 게이트로 교육 품질을 보증합니다. 모집 미달·중도 이탈·기기 장애·강사 불참·협력학교 미확보 등 5대 리스크에 대해 사전 예방 + 발생 시 대응의 이중 안전망을 구축합니다.
