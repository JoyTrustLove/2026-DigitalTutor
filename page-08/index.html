3 학생 개별 기기 로그인·앱 설정 보조 {확인필요} 계정·앱 관리 실무 역량 교육 강화
4 수업 후 학습 데이터 정리·리포트 보조 {확인필요} 수업 '후' 데이터 관리 역량도 포함 필요

1 실제 학교 현장에서의 실습 경험 {확인필요} → 현장체험 2차시 신설로 직접 대응
2 시도별 플랫폼·기기 특화 교육 {확인필요} → 지역특화 25차시 이원화 설계로 대응

8면 핵심 요약
2026년은 AI for All 정책 전환으로 AI 디지털교과서가 학교 현장에 본격 적용되는 원년입니다. 서울은 디벗·MDMS·서울런4.0, 인천은 에듀아이·AI 코스웨어·스마트기기라는 서로 다른 인프라 환경에서 디지털튜터가 실무를 수행해야 합니다. 2025년 실태조사에서 교사와 기 수료 튜터 모두 '현장 맞춤 실습'과 '시도별 특화 교육'을 가장 필요한 요소로 꼽았으며, 이는 본 컨소시엄의 공통 110차시 + 지역특화 25차시 + 현장체험 2차시 설계 근거로 직결됩니다.
